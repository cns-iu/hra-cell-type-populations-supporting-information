In HRApop v1.0, 16,293 datasets served as Input for the RUI2CTpop Workflow, and they cover 49 organs. 662 datasets make up the HRApop Atlas. 5,672 datasets have an extraction site but no cell type population. Inversely, 6,395 datasets have a cell type population but no extraction site. 3,564 datasets have neither. The total number of non-atlas datasets is 15,631. The Sankey diagrams below offer an overview of HRApop provenance.
